鹊桥仙

2014-03-11
诗词 词

无情流水，连天芳草，黯淡无言南浦。
人生毕竟一相逢，对花落、不成凄苦。

慢行也远，相思也老，轻别轻分几度。
密云翻涌始回头，一霎那、狂风暴雨。

2014.3.11 南京
